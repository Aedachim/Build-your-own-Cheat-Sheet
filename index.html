Table tags

Tag Name Description
<table> Table Summary of all table tags
<thead> thead Contains of the header in the table element
<tbody> tbody Contains of the body in table element
<tfoot> tfoot Contains of the footer in the table element
<tr> table row Shows the row in the table element
<td> table data Shows the data in the table element



Table attributes

Attributes Name Description
<scope> scope(col/row) The scope attribute can take one of two values: "row"/"col" = makes it clear that the heading is just for a row/column
<colspan> colspan Data can span columns using colspan
<rowspan> rowspan Data can span multiple rows using rowspan
